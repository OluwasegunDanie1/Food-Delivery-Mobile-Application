#  Flutter Food Delivery App

A modern food delivery mobile application built with Flutter.
This app allows users to browse restaurants, view menus, add items to cart,
and place food orders.

##  Features
- User authentication
- Browse restaurants & menus
- Add to cart
- Order summary
- Clean and responsive UI
- Flutter best practices

##  Tech Stack
- Flutter
- Dart
- Provider (State management)
- Sharedpreference (local storage)
- Firebase (optional)
